# Demo Assets

## RecallDoc_Demo.pdf

This directory should contain a demo PDF file named `RecallDoc_Demo.pdf` that will be bundled with the app.

### Requirements:
- File name: `RecallDoc_Demo.pdf`
- Content: 2-3 pages of educational content (e.g., about spaced repetition, learning techniques, or app features)
- Size: Keep under 1MB for bundle size
- Text-based: Should contain selectable text for the demo to work properly

### Suggested Content:
Page 1: Introduction to Spaced Repetition
- What is spaced repetition?
- Benefits of spaced repetition
- How it improves long-term memory

Page 2: Using RecallDoc
- Creating sections from your reading
- Generating flashcards
- Reviewing with the SRS algorithm

Page 3: Study Best Practices
- Consistent daily reviews
- Breaking down complex topics
- Using different card types
